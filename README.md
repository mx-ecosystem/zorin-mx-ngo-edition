## English Version

### About the Project
This project is a deeply optimized, stripped-down legacy version of Zorin OS, custom-built for high efficiency, system logic, and maximum performance on older or budget-friendly hardware. Originally designed as a commercial prototype, the project has now been fully opened to the public and transitioned to an open-source model.

If you are looking for a highly responsive, clean, and pre-configured system that values your hardware resources, this build is for you.

### Key Features & Optimizations
*   **Bloatware Removed:** Completely stripped of telemetry, unnecessary background services, and system heavyweights.
*   **System Logic over "Empty Sandbox":** Everything is built for efficiency, clean structures, and immediate out-of-the-box readiness.
*   **Optimized Resource Usage:** Extremely low RAM footprint on idle, making it perfect for budget setups (e.g., older Ryzen/Intel machines).
*   **Visual Order:** Aesthetic, consistent desktop layout designed to keep you focused on your work.

## Wersja Polska

### O projekcie
Ten projekt to głęboko zoptymalizowana, lekka wersja systemu Zorin OS, stworzona z myślą o maksymalnej wydajności, logice systemowej i szybkości działania na starszym lub budżetowym sprzęcie. Projekt pierwotnie powstawał jako prototyp komercyjny, jednak ostatecznie zdecydowałem o udostępnieniu go w pełni publicznie i bezpłatnie dla społeczności open-source.

Jeśli szukasz niezwykle responsywnego, czystego i gotowego do pracy systemu, który szanuje zasoby Twojego komputera – ten projekt jest dla Ciebie.

### Kluczowe modyfikacje i optymalizacje
*   **Usunięty bloatware:** Całkowicie wycięta telemetria, zbędne usługi działające w tle oraz zasobożerne aplikacje.
*   **Logika zamiast chaosu:** Każdy element systemu został skonfigurowany pod kątem maksymalnej stabilności i użyteczności od pierwszego uruchomienia.
*   **Optymalizacja zasobów:** Minimalne zużycie pamięci RAM w spoczynku, idealne dla słabszych konfiguracji sprzętowych.
*   **Wizualny porządek:** Estetyczny, minimalistyczny i spójny układ graficzny pulpitu, który sprzyja skupieniu.
